Commands

  1. npm install

    2. This command will install all the dependencies listed in the package.json file and the node_modules.

  2. node file_name.js

    3. This command will run our js file

  3. nodemon file_name.js

    4. This command will also run our js file. The difference with the previous one is that nodemon will automatically restart the node application when there are new changes to it. So we don't need to stop and rerun the program every time we have changed something.

  4. npx

    5. We use npx for running a package without installing it.

  5. npx budo index.js —live —open

    6. budō allows you to get your scripts up and running quickly in a local environment. After running this command a window will open that displays our app. The index.js file is usually connected to an html file. budo i focused on incremental reloading, LiveReload integration (including CSS injection) and other high-level features for rapid prototyping.

  6. npm init

    7. npm init can be used to set up a new or existing npm package.
