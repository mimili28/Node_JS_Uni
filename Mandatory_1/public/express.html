What is Express

"Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications."

Express.js helps you manage everything, from routes, to handling requests and views.

  • Allows to dynamically render HTML Pages based on passing arguments to templates.
  • you can use a variety of choice for templating language
  • you can use a database to provide a backend for your node.js application

With Express you don't have to repeate same code over and over again and it saves you a lot of time compared to only using plain Node.js.
