JavaScript Strings

A JavaScript string is zero or more characters written inside quotes.

JavaScript strings are used for storing and manipulating text.

Initializing variable with string value: 

var day = "Sunday"

Convert Strings to Numbers

  • parseInt()
  • parseFloat()
  • Number()
    • Number('123'); // returns 123

String length

var text = "something";
var length = text.length;

Escape Characters

  • \' -> Single quote
  • \" -> Double quote
  • \\ -> Backslash

Finding a String in a String

The indexOf() method returns the index of (the position of) the first occurrence of a specified text in a string:

  • var string = "Find string in a string";
    • var pos = str.indexOf("string");

    The lastIndexOf() method returns the index of the last occurrence of a specified text in a string:

    var str = "Find string in a string";
    var pos = str.lastIndexOf("string");

    Both indexOf(), and lastIndexOf() return -1 if the text is not found.

    Extracting String Parts

    There are 3 methods for extracting a part of a string: 

    slice(start, end)

    If a parameter is negative, the position is counted from the end 

    substring(start, end)

    substring() and slice() are similar with the difference that substring() cannot accept negative indexes

    substr(start, length)

    Replacing String Content

    The replace() method replaces a specified value with another value in a string: 

    "Good morning!"
    str.replace("morning", "afternoon");

    By default, the replace() method replaces only the first match:

    Converting to Upper and Lower Case

    To convert a string to upper case we use toUpperCase(): 

    var text1 = "Hello World!";
    var text2 = text1.toUpperCase();

    A string is converted to lower case with toLowerCase()
