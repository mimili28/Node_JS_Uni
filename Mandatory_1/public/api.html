Node.js REST API

API

API stands for aplication programming interfaces. It is a set of rules that allow programs to talk to each other.

REST

REST is abbreviation REpresentational State Transfer. It determines how the API looks like, it is a set of rules to be followed when creating an API.

It is used to access and manipulate data using several stateless operations, integral to the HTTP protocol:

  • GET (retrieve an index of resources or an individual resource)
  • POST (create a resource or generally provide data)
  • PUT (create or replace a resource)
  • PATCH (update/modify a resource)
  • DELETE (remove a resource)

By using these operations and a resource name as an address we can create our REST APIs.

Within Node.js, we can use JavaScript to build our REST APIs and our software
