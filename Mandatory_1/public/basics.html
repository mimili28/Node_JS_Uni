What is Javascript

JavaScript ("JS" for short) is a dynamic programming language that, when applied to an HTML document, can provide dynamic interactivity on websites.

Variables

Variables are containers that you can store values in.

Declaring a variable: 

let myVariable;

Giving value to a variable after it has been declared: 

myVariable = 'John';

Declaring and giving value in one line: 

let myVariable = 'John';

You can retrieve the value by just calling the variable by name: 

myVariable;

You can change the given value of a variable: 

let myVariable = 'John'
myVariable = 'Mike'

Variables can hold values that have differnet data types sush as String, Number, Boolean, Array and Object

Scope

Scope defines where variables and functions are accessible inside of our program. In JavaScript, there are two kinds of scope - global scope, and function scope.

  • function scope - variable is only accessible inside of the function it was created in or, any nested functions.
  • global scope - the variable can be accessed by all scripts and functions on a web page.

    • Var, let and const - the difference

    var vs let

  • var
    • function scoped
    • undefined when accessing a variable before it's declared
  • let
    • block scoped -> a variable created with the let keyword is available only inside the “block” (anything surrounded by a curly brace {}) that it was created in as well as any nested blocks.
    • ReferenceError when accessing a variable before it's declared

    • let vs const

  • const is almost exactly the same as let. However, the only difference is that once you’ve assigned a value to a variable using const, you can’t reassign it to a new value.

    • Printing to the console

    console.log() - a function in JavaScript which is used to print any kind of variables defined before in it or to just print any message that needs to be displayed to the user.

    Example 

    myName = 'Maria'
    console.log("My name is ", myName)

    will return to the console:

  • My name is Maria
