What is jQuery?

JQuery is a lightweight JavaScript library that makes it much more easier to use JavaScript on our web application Some functions require many lines of code in JavaScript but with jQuery these lines are wrapped together in a method that can be called with a single line of code.

jQuery also simplifies a lot of the complicated things from JavaScript.

The jQuery library contains the following features:

  • HTML/DOM manipulation
  • CSS manipulation
  • HTML event methods
  • Effects and animations
  • AJAX

Using jQuery

In order to use jQuery we need to first either install it or include it from a CDN (Content Delivery Network).

We should go to Google Hosted Libraries and copy the current jquery version script.

We then paste the script in the html file. The script tag should be either at the end of the body or inside the head section.

jQuery syntax:

$(selector).action()

The selector is a html element and the action is what we want to do with the element

Example:

$("a").hide() // hides all a elements.

The document ready event

We put our methods in a document ready event in order to have our jQueties after the page has loaded.

$(document).ready(function(){ 
        // jQuery methods go here 
      });
