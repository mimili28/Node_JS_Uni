npm

What is npm

1. It is an online repository for the publishing of open-source Node.js projects

2. It is also a software Package Manager and Installer

Installing npm

npm is beeing installed together with Node.js so in order to use npm first you need to install Node.js

Node.js can be installed from https://nodejs.org

Software Package Manager

All npm packages are defined in files called package.json.

We write the packages in a key called dependencies as key-value pairs where the key is the package name and the value is the version.

Example:

  • {
    "dependencies": {
    "express": "4.17.1",
    "nodemon": "^2.0.2",
    "request": "2.88.2"
    }
    }
