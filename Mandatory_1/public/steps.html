How to create our API:

1. We need to create the package.json file

More information about it: npm section

2. npm install

In the terminal we need to locate the folder where we have our package.json and run the command: npm install. This command will install all the dependencies in the package.json file and the node_modules.

3. Create our .js file(app.js)

Now in the app.js file we need to import the express package:

  • const express= require("express");
    const app=express();

    • or in one line:

  • const app= require("express")();

    • 4. Finally we have to connect to the server:

  • app.listen(3000)

    • This means that our program will run on port 3000

    After we have created the foundations of our web application we can start impelementing different functionalities
