How to create our API:

1. We need to create the package.json file

More information about it: npm section

2. npm install

In the terminal we need to locate the folder where we have our package.json and run the command: npm install. This command will install all the dependencies in the package.json file and the node_modules.

3. Create our .js file(app.js)

Now in the app.js file we need to import the express package:

const express= require("express"); 
        const app=express();

or in one line:

const app= require("express")();

4. Finally we have to connect to the server:

app.listen(3000)

This means that our program will run on port 3000

After we have created the foundations of our web application we can start impelementing different functionalities

REST API operations

Now we need to connect routes to http methods in order to be be able to see something in our web application

Routing

GET

We use app.get() method to get requests

Example:

app.get("/hello", (req, res) => { //req is for request object and res for response object
        res.send({message: "Hello there"}); 
      });

What these line of code mean is that when we go to route "localhost:3000/hello" we will see a message "Hello there"

res.send() is a response method for sending a response of various type. It is the last line of code we should have in our method, if there are lines after it they will simply not run

Route parameters

Route parameters are URL segments used to capture the values that are specified at their position in the URL. The captured values are populated in the req.params object, with the name of the route parameter specified in the path as their respective keys.

app.get("/user/:id",(req,res) => { 
        return res.send({id: req.params.id}); 
      });

In this case the route parameter is "id"

Route path: /user/:userId

Request URL: http://localhost:3000/user/5

req.params: { "userId": "5" }
